# 数据分析学习资料专区
本专区记录我的**系统化数据分析学习路径**，聚焦“从基础工具到业务实战”的能力闭环，明确学习边界（只覆盖数据分析核心技能），同时展示学习内容与落地成果。


## 学习定位与边界
- **核心定位**：围绕「数据分析师岗位核心技能」展开，拒绝无目的泛学，所有学习内容均服务于“解决业务问题、落地分析项目”。
- **学习边界**：
  ✅ 聚焦：SQL/Python/Tableau等工具实操、业务分析方法（漏斗/分群/A/B测试）、数据建模基础（回归/分类）；
  ❌ 不涉及：纯算法开发（如深度学习）、底层技术开发（如数据库搭建）。


## 📚 学习路径与内容分类
我的学习按“**基础工具→分析方法→业务实战**”递进，对应资料如下：

### 1. 基础工具：从“会用”到“用熟”
覆盖数据分析必备工具的实操技巧，配套练习代码+速查表，确保工具能力扎实。
- **SQL**：
  - 资料：个人总结资料；
  - 核心内容：窗口函数、自连接、子查询、性能优化（适合电商/用户行为分析）。
- **Python**：
  - 资料：个人总结资料；
  - 核心内容：数据读取/清洗/聚合、可视化（Matplotlib/Seaborn）、基础建模库（Scikit-learn）。
- **可视化工具**：
  - 资料：《Tableau交互式仪表盘技巧.md》《Excel数据透视表进阶笔记.md》；
  - 核心内容：维度/度量搭配、参数筛选、联动图表（适配业务指标监控场景）。


### 2. 分析方法：从“懂概念”到“会落地”
聚焦业务分析的通用方法，配套案例拆解，确保能把方法用在真实项目中。
- 资料：《用户增长分析方法总结.md》《A/B测试评估指南.md》；
- 核心内容：
  - 漏斗分析：拆解用户转化链路（如电商“浏览→加购→下单”）；
  - 用户分群：RFM模型、分层运营（如高价值用户留存策略）；
  - A/B测试：指标设计、显著性检验（如按钮颜色对点击的影响）。


### 3. 业务场景：从“学理论”到“练实战”
结合互联网/电商/用户增长等高频场景，将工具+方法落地，衔接核心项目。
- 资料：《电商销量预测案例教程.md》《用户流失原因拆解笔记.md》；
- 核心内容：
  - 把“线性回归模型”用在电商销量预测；
  - 用“多维度分群”定位用户流失原因；
  - 所有练习内容均与仓库`projects`中的实战项目对应（学练结合）。
