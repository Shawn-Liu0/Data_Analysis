## 核心项目展示
### 项目1：电商平台用户留存率分析与优化建议
- **工具**：SQL（数据提取） + Tableau（可视化） + Excel（漏斗拆解）
- **业务背景**：某电商平台30日留存率环比下降8%，需定位流失原因并给出解决方案
- **核心分析**：
  1. 通过SQL拆分用户生命周期，发现“新用户首单后7天内未复购”是核心流失节点（占比62%）；
  2. 用Tableau做用户分群，定位“低客单价（<50元）+ 非包邮用户”留存率仅为12%（远低于均值35%）；
- **落地建议**：针对低客单价新用户推送“首单后7天内满50减5包邮券”；
- **效果（模拟）**：方案落地后，目标用户30日留存率提升15%。
- **项目文件**：[用户留存分析报告](projects/电商用户留存分析/留存分析报告.pdf) | [Tableau可视化源文件](projects/电商用户留存分析/留存分析.twb)


### 项目2：短视频App用户行为路径转化漏斗
- **工具**：SQL（日志数据提取） + Excel（漏斗计算）
- **业务目标**：优化从“首页→内容播放→互动（点赞/评论）→关注”的转化链路
- **核心结论**：“内容播放→互动”环节流失率达58%，原因是“互动按钮入口隐蔽”；
- **项目文件**：[转化漏斗分析表](projects/短视频用户路径/转化漏斗.xlsx) | [SQL提数脚本](projects/短视频用户路径/用户行为提数.sql)
