# 关于项目
- 此项目是一个练手项目，主要针对银行用户的流失进行分析
- 在这个项目中，我的目标是：
    - **熟悉excel数据透视表、数据透视图、常见函数使用**
    - **熟悉Tableau绘图**
    - **复习决策树模型及模型调参**
    - 进行业务解读（简略）

# 关于数据集
- 本数据集来源于阿里天池，链接：https://tianchi.aliyun.com/dataset/207617
- 数据集字段表
  
|   列序	|字段名（英文）	|中文含义	|数据类型	|说明|
|----------|----------|----------|----------|----------|
|1  |RowNumber	|行号	|int	|纯索引，无业务意义，建模时可删除|
|2	|CustomerId	|客户唯一ID	|int	|用于标识客户，建模时通常不作为特征|
|3	|Surname	|客户姓氏	|string	|一般不用于建模（除非做 NLP 特征）|
|4	|CreditScore	|信用评分	|int	|越高信用越好（范围约 300–850）|
|5	|Geography	|客户所在国家	|string	|取值：France, Spain, Germany|
|6	|Gender	|性别	|string	|Male / Female|
|7	|Age	|年龄	|int	|客户年龄（岁）|
|8	|Tenure	|在本行开户年限	|int	|单位：年（0 表示新开户）|
|9	|Balance	|账户余额	|float	|当前账户存款金额（欧元）|
|10	|NumOfProducts	|持有产品数量	|int	|如储蓄账户、信用卡、理财等（通常 1~4）|
|11	|HasCrCard	|是否持有信用卡	|int	|1 = 是，0 = 否|
|12	|IsActiveMember	|是否为活跃会员	|int	|1 = 是（如常登录网银、交易），0 = 否|
|13	|EstimatedSalary	|预估年薪	|float	|客户年收入（欧元）|
|14	|Exited	|是否流失（目标变量）	|int	|1 = 已流失（关闭账户），0 = 仍留存|
